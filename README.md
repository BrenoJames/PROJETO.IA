# PROJETO.IA
ste projeto utiliza **Inteligência Artificial** para prever o preço de uma pizza baseado no seu diâmetro. É um exemplo prático de **Regressão Linear** aplicada a um problema do dia a dia.  O aplicativo possui uma interface interativa onde você pode arrastar uma barra para definir o tamanho da pizza e ver o preço sugerido pela IA em tempo real.
